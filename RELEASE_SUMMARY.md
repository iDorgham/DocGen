## 🚀 Release Commands

### Installation
```bash
pip install docgen-cli
```

### Basic Usage
```bash
# Create a new project
docgen create my-awesome-project

# Generate documentation
docgen spec
docgen plan
docgen marketing

# Validate project
docgen validate

# Get help
docgen help
```
